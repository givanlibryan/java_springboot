PRODUCT MODULES

Pada sesi assignment 3 membuat RESTFUL API untuk modul produk :

Dimana terdiri dari :
A. nama,
B. hargaBeli
C. hargaJual

Dengan Operation POST , GET & DELETE.

Implementasi program menggunakan dependency sebagai berikut:
	> Spring Boot DevTools 
	> Spring Web 
	> Spring Data JPA
	> MySQL Driver
	> Bootstrap
	> Thymeleaf

Spring menggunakan version 2.7.0

Java menggunakan version 17

Menggunakan Bootstrap dan Thymeleaf karena menambahkan landing page  

akses landing page di : "localhost:8080"

akses data di : "http://localhost:8080/assignment/products"
